The use of UiPath workflows requires installed UiPath Studio - https://www.uipath.com/product/studio

The dependencies of UiPath workflows are defined in 'project.json'. In case of separating workflows from 'project.json' make sure you download these dependencies in Studio via 'DESIGN' - 'Manage packages'.

For functional ROSSUM's AI data extraction service requires to have valid credantials json (credentials.json) - https://rossum.ai/contact
